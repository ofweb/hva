# Rust Docs

Use the Rust MCP tools first for Rust questions.

- `crates_search` for finding crates
- `crates_package_info` for crate versions and metadata
- `crates_dependencies` for dependency trees
- `docs_rs_read` for docs.rs pages

Use this for:

- latest crate version
- crate metadata
- docs.rs pages
- feature flags
- dependency trees
- "what crate is this?"
- "what version is current?"

Use `crates_package_info` first when the user asks for the latest version.

Never guess Rust crate versions.

Do this before broad web search.
