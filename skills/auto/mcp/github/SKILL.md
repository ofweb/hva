# GitHub

Use GitHub MCP tools for upstream repo lookup.

- `github_search_code` for code search
- `github_read_file` for file contents
- `github_search_repositories` for repo search
- `github_list_directory` for repo directories

Use this before broad web search.

Use it when the answer is likely in a GitHub repo, not a local workspace file.

Use this for:

- upstream code search
- file lookup
- PR/issue lookup
- branch or commit context
- "where is this file in the repo?"
