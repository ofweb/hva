# npm Search

Use npm MCP tools for npm package questions.

- `npm_package_search` for finding packages
- `npm_package_info` for versions and metadata

Use this for:

- latest package version
- package metadata
- package search by keyword
- "what npm package does X?"
- "which version is current?"

If the exact package name is known, use `npm_package_info`.

If the package name is fuzzy, use `npm_package_search` first.

Never guess npm package versions.

Use these before broad web search.
